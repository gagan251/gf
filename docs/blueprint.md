# **App Name**: Bharat Learning Hub

## Core Features:

- User Authentication: Secure student and admin login using Firebase Authentication.
- Course Listing: Display available courses (Typing, Speed Building, Stenography) with descriptions and enrollment buttons.
- Student Dashboard: Personalized dashboard for students with access to lessons and tests after login.
- Admin Panel: Admin interface for uploading lessons, tests, and managing student accounts.
- Content Management: Store and manage course content, including lessons, tests and recordings in Firebase Database.
- Responsive Design: Fully responsive layout for seamless access on mobile and desktop devices.
- AI-Powered Skill Assessment: Generative AI tool assesses user performance through test data and offers custom learning paths and skill enhancement plans.

## Style Guidelines:

- Primary color: Blue (#007BFF) for a professional and trustworthy feel.
- Background color: Light grey (#F8F9FA) to maintain a clean and minimal aesthetic.
- Accent color: A slightly darker shade of blue (#0056b3) for button hovers and interactive elements.
- Body font: 'Inter', a sans-serif font for a modern, neutral, and readable look in both headings and body text.
- Note: currently only Google Fonts are supported.
- Card-based layout with rounded corners and soft shadows to provide a modern and visually appealing structure.
- Consistent spacing and visual hierarchy throughout the website to ensure ease of navigation.
- Clean and professional icons for menu items, course sections, and footer information.
- Subtle hover effects and transitions to enhance user interaction and engagement.
